# Desafio Alura - JavaScript

## Sobre o projeto
Este projeto foi desenvolvido durante minha primeira aula de JavaScript na Alura.

## Funcionalidades
- Exibe mensagem de boas-vindas
- Mostra alerta de erro
- Captura o nome do usuário
- Captura a idade do usuário
- Verifica se o usuário pode tirar habilitação

## Tecnologias utilizadas
- HTML
- JavaScript

## Objetivo
Praticar conceitos básicos de JavaScript, como:
- Variáveis
- Alert
- Prompt
- Estruturas condicionais (if)

## Como executar o projeto
1. Baixe os arquivos do repositório
2. Abra o arquivo `index.html` no navegador
